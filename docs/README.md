# Ecommerce Data Engineering Project

## Objective
Build an ETL pipeline that processes ecommerce sales data and generates business insights.

## Dataset
- customers.csv
- orders.csv
- products.csv

## Pipeline Steps
1. Extract CSV data
2. Inspect datasets
3. Validate data quality
4. Join datasets
5. Calculate revenue metrics
6. Aggregate customer revenue
7. Aggregate product revenue
8. Export analytics datasets

## Outputs
- full_data.csv
- customer_revenue.csv
- product_revenue.csv
